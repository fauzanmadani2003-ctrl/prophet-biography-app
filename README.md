# Prophet Biography Application

This application is dedicated to providing a comprehensive biography of Prophet Mohammed (PBUH). It includes reliable sources and documents to ensure accurate representation of his life and teachings.

## Features

- **Chronological Timeline**: View significant events in the life of Prophet Mohammed (PBUH) through an interactive timeline.
- **Biography Cards**: Explore detailed biographical information presented in a user-friendly card format.
- **Source Documentation**: Access reliable documents such as "Raheeq-al-Maqtoom" and "Seeratunnabi" to deepen your understanding of his life.
- **Responsive Design**: The application is designed to be responsive and accessible on various devices.

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd prophet-biography-app
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To start the application, run:
```
npm start
```
This will launch the application in your default web browser.
